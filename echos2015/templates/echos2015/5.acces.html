{% if lang == "fr" %}Du Saix jusqu'à la Ferme du Faï{% else %}From le Saix to le Faï farm{% endif %}

{% if lang == "fr" %} Une fois arrivés au village du Saix, rejoignez le festival à pieds en suivant le parcours fléché.

{% if lang == "fr" %} Une navette sera mise en place à partir du parking pour vous décharger de vos affaires. {% else %} A shuttle will carry your luggage from the parking to the farm. {% endif %}
